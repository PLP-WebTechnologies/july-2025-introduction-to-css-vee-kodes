CSS Basics

CSS (Cascading Style Sheets) is a language used to style HTML elements for visual appeal. While HTML creates the structure of a webpage, CSS controls:

  • Colors and fonts
  • spacing and positioning
  • Borders and backgrounds
  • Responsive layouts

Connecting CSS to HTML

For this assignment, I'm using an external stylesheet. This means:

  1. All CSS rules live in a separate file (style.css)
  2. The HTML file links to it in the head section
  3. This approach keeps my code organized and reusable

The CSS Box Model

Every HTML element is a box with four layers:

  1. Content: The actual text or media.
  2. Padding: Space inside the border.
  3. Border: The visible edge around padding.
  4. Margin: Space outside the border.
